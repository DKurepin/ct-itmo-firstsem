3. mod-sort.zip. Отсортируйте файлы в директории по возрастанию времени модификации. Посчитайте шестнадцатиричный SHA-256 хеш конкатенации данных этих файлов в этом порядке (без каких-либо дополнительных символов).
#!/bin/bash
touch file | ls -S -t | cat $(ls -r -t) > file | sha256sum file
Answer(6090cc6a2841c0a8def75d220f01692c5d024cc8ab2238c75f95e9ed726e7499)

4. phone-numbers. Посчитайте количество телефонных номеров в формате E.164 в данном файле.
#!/bin/bash
grep -o -w "+\{0,1\}[1-9][0-9]\{1,14\}" phone-numbers | wc -l
Answer(50171)

5. wget-pdfs. Скачайте все файлы по указанным ссылкам. Посчитайте суммарный размер PDF-документов среди скачанных файлов в байтах.
#!/bin/bash
wget -i wget-pdfs -P links; du -bc `grep -ra "%PDF" links/* | cut -d ":" -f 1`

6. grep-logs.zip. Даны логи HTTP-сервера. Найдите все строки с ошибками сервера. В них ответ сервера имеет вид HTTP/1.1 5xx, где xx — любые две цифры. В каждой такой строке выберите первое слово после даты — метод запроса. Найдите все различные такие слова и выведите их через пробел в лексикографическом порядке. Важно! Запрещается в явном виде распаковывать архивированные логи.
#!/bin/bash
zgrep "HTTP/1.1 5" logs/* | cut -d "/" -f2 | cut -d "]" -f2 | tr -s " " | sort | uniq
